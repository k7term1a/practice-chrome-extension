# Chrome extension 練習
[相關筆記](https://hackmd.io/@kaichi/SJGqL2UKZx)

## 目的
以盡量不使用 vibe coding 方式，實作一個用於瀏覽器新分頁的擴充套件。<br>預期效果有：
1. 新分頁背景
    - [x] 按鈕美化，並在一般情況隱藏。
    - 替換成不同圖片或影片
        - [x] 上傳不同影片並套用
        - [x] 可以上傳 `mp4`、`jpg`、`jepg`、`png` 了
    - 設定多個影像，隨機或順序顯示
2. 書籤排列
    - 使用者自行排序，或其他條件排序
    - 可選擇是否在側邊欄出現
3. 附加功能
    - 指定期間出現設定好的捷徑
    - 快節儉組合開啟指定捷徑
